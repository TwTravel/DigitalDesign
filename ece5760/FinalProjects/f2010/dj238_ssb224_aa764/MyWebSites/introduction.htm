Introduction

Virtual paint is a system which can be used to draw colored pictures on any plain surfaces like a wall using hand movements and fingers ( with colored bands). This project draws its motivation from the sixth sense project done at MIT media labs where using different gesture detection and color segmentation techniques a software running on a handheld computing device performs a variety of different tasks like making notes in air, reading a news paper with projected videos, capturing photographs using hand actions and even painting on walls. Intrigued by the scope of the project we decided to do something similar in hardware. so within the constraints of availability of hardware and time we implemented Virtual Paint.
