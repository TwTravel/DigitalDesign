Documentation

First unzip the Lab5.zip file, which has had all the intermediate files removed. Launch Quartus II, compile the circuit, and download onto the DE2 board.

Currently, the switches and buttons are used to control simple operations. SW[17:16] is used to control which object is generated. Use 0b00 for a circle and 0b10 for a cube. SW[9:0] is used to indicate the step size for the circle generator, in units of . KEY[3] is used to reset the whole pipeline and the screen buffer.

The eight seven segment displays display the four points that were last entered into the Bresenhem module.
