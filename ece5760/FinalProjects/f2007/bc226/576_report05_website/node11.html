Testing and Verification

Most of the modules were developed without downloading the code onto the DE2 board. Thus, the source code has many $display and $finish calls embedded throughout the circuit.

Additionally, since the number representation is 16-bit fixed-point, several helper C programs were written generate the sine and cosine tables and to ensure that other portions of the Verilog were functioning correctly. These programs have been included in the source zip file.
