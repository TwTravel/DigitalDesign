Library Overview

The Quartus Prime software installation includes the Intel FPGA IP library (1). The library alllows you to specify in detail how your Verilog should map to specific hardware on the FPGA. For instance, there is support for several types on memory which can be mapped to Logic Elements, or MLAB, or M10K blocks. You can find the installed IP Catalog on the menu ( Tools âž¤ IP Catalog ) to efficiently parameterize and generate synthesis and simulation files for your custom IP variation (2). Each different IP core has a users guide with all the details you need to make it work. Generally, you navigate a series of dialog boxes to set up the core, then include the generated module in your project, and instatiate it in the top-level module.
The Intel FPGA IP library includes the following types of IP cores:

  • Basic functions
    • Arithmetic
      • CORDIC
      • MAC
      • floating point, including many functions
      • complex multiply -- config dialog 1, config dialog 2, config dialog 3, config dialog 4,
    • Clocks and PLLs
    • Memory -- users guide
      • RAM
      • 2-port RAM -- config1, 2, 3, 5, 6, 7, 10
      • ROM,
      • shift registers,
      • FIFO
  • DSP functions -- seems to duplicate Qsys functions.
  • Interface protocols -- seems to duplicate Qsys functions.
  • Low power functions -- seems to duplicate Qsys functions.
  • Memory interfaces and controllers -- seems to duplicate Qsys functions.
  • Processors and peripherals -- seems to duplicate Qsys functions.
